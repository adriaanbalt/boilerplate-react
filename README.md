# Boilerplate to build React

## Installation

```
$ npm install
$ npm start
```

Go to `localhost:3000`

## Goals

* Create a new page
* Load new page via router with unique URL
* Create component
* Load component onto new page
* Component must receive data from `DATA.json`

### Extra Credit

* Component uses an array of data with a `map()`

### Requirements
* Node: 4.2.1
* NPM: 2.14.7
